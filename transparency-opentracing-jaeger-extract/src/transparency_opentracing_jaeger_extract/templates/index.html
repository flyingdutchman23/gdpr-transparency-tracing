This is our simple service to extract information about personal data processing from Jaeger. The api is available below api/*. Currently these services are available:

  • api/purposes
  • api/categories
  • api/origins
  • api/recipients
  • api/durations
  • api/automated
  • api/3rdparty
  • api/all
  • api/groups
  • api/groups/no-duplicates
